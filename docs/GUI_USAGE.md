# 🚀 Руководство по HH Auto Apply - Desktop GUI

## ✨ Введение

**HH Auto Apply** — это приложение для автоматизации отправки откликов на вакансии на сайте hh.ru. 
Оно:
- 🔍 Автоматически ищет вакансии по вашим критериям
- 📝 Генерирует персонализированные сопроводительные письма с помощью AI
- 📤 Отправляет отклики в один клик
- 📊 Ведет статистику и историю

## 🎯 Быстрый старт

### 1️⃣ Запуск приложения

**Windows (exe файл):**
```
Двойной клик на HH_Auto_Apply.exe
```

**Или из Python:**
```
python run_gui.py
```

### 2️⃣ Первичная настройка

При первом запуске перейдите на вкладку **"⚙️ Конфигурация"** и заполните:

#### 📍 Параметры поиска
- **Поисковый запрос** - Что ищут (например: "Python разработчик")
- **ID регионов** - Оставить пустым для всех, или указать ID (1=Москва, 2=СПБ)
- **Только удаленная работа** - Галочка если нужна удаленка
- **Макс. откликов** - Сколько откликов отправить за сеанс (например: 100)

#### 📋 Резюме
- **Название резюме (маска)** - Какое резюме использовать (например: "Python разработчик")
- **Ошибка если резюме не найдено** - Если заполнить, приложение вернет ошибку если резюме не найдено

#### 💌 Сопроводительное письмо
- **Файл письма** - Путь к файлу cover_letter.txt
- **Обязательно требовать письмо** - Галочка если письмо необходимо

#### 🤖 AI - Генерация писем (необязательно)
Если хотите использовать AI для автоматической генерации писем:

1. **Включите AI генерацию** - Поставьте галочку
2. **Получите API ключ:**
   - Зарегистрируйтесь на https://openrouter.ai
   - Перейдите в Keys и скопируйте ваш ключ
   - Вставьте в поле "OpenRouter API ключ"
3. **Выберите модель** - Рекомендуемые бесплатные:
   - `mistralai/mistral-7b-instruct:free` - Легкая, быстрая
   - `google/gemma-7b-it:free` - Хорошее качество
4. **Отредактируйте промпт** - Инструкция для AI как писать письма

#### 🌐 Браузер
- **Headless режим** - Браузер запустится без визуального окна (быстрее)
- **Мин./Макс. задержка** - Пауза между действиями в секундах (5-7 рекомендуется)

#### 💾 Сохранение
Нажмите **"💾 Сохранить"** когда готовы!

### 3️⃣ Запуск процесса

Перейдите на вкладку **"▶️ Запуск"**:

1. **Dry-run режим** (опционально)
   - Галочка "📋 Dry-run" для тестирования БЕЗ реальных откликов
   - Полезно для проверки перед настоящим запуском

2. **Запустить приложение**
   - Нажмите кнопку **"▶️ Запустить"**
   - Браузер откроется (он войзет на hh.ru и будет собирать вакансии)
   - Первый раз нужно через браузер войти в свой аккаунт на hh.ru
   - Введите логин/пароль в открытом браузере

3. **Наблюдайте за процессом**
   - **Логи** - Все действия выводятся в реал-тайм
   - **Статистика** - Видите сколько найдено, открыто, откликнулось и ошибок

4. **Остановить**
   - Нажмите **"⏹️ Остановить"** чтобы завершить в любой момент

### 4️⃣ История откликов

Перейдите на вкладку **"📊 История"**:
- Таблица всех отправленных откликов
- Заголовок вакансии и ссылка
- Автоматически обновляется каждые 5 секунд

## ❓ Часто задаваемые вопросы

### Q: Приложение зависает?
A: Попробуйте:
- Увеличить задержки (мин./макс. в браузере)
- Запустить в headless режиме
- Проверить интернет соединение

### Q: Почему не отправляются отклики?
A: Проверьте:
- Вы залогинены на hh.ru в браузере?
- Верно ли указаны ID резюме?
- Заполнено ли сопроводительное письмо?
- Нет ли ограничений по количеству откликов в день на hh.ru?

### Q: Как использовать AI?
A: 
1. Создайте аккаунт на https://openrouter.ai (бесплатно)
2. Возьмите API ключ из раздела Keys
3. Впишите в конфиге
4. Включите AI в галочке
5. Отредактируйте prompt.txt как нужно

### Q: Как сохранить разные конфигурации?
A: На будущее! Сейчас сохраняется в .env файл. Вы можете копировать .env файл чтобы создавать разные профили.

### Q: Это безопасно?
A:
- ✅ Ваши пароли НЕ сохраняются
- ✅ API ключ сохраняется шифрованным
- ✅ Отклики отправляются с вашего браузера (hh.ru видит это как вас)
- ✅ Нет шпионажа - все работает локально

## 🔧 Настройки Advanced

### Переменные окружения (.env файл)

Для опытных пользователей - прямое редактирование .env файла:

```env
# Основное
HH_SEARCH_QUERY="Python разработчик"
HH_REGION_IDS="1,2"        # 1=Москва, 2=СПБ, пусто=везде
HH_MAX_APPLIES="100"       # Сколько откликов отправить
HH_RESUME_TITLE_MATCH="Python разработчик"

# Браузер
HH_HEADLESS="true"         # true/false
HH_MIN_SLEEP="3"
HH_MAX_SLEEP="7"
HH_PERSIST_DIR=".hh_user"  # Папка для сохранения сессии браузера

# AI
HH_USE_AI_COVER_LETTER="true"
OPENROUTER_API_KEY="sk-or-v1-..."
AI_MODEL="mistralai/mistral-7b-instruct:free"
```

## 📦 Установка зависимостей

### Из exe файла
Просто запустите - всё уже включено!

### Из исходного кода
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Zakirov-Yuriy/hh_auto_apply.git
cd hh_auto_apply

# 2. Создайте виртуальное окружение
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Установите браузер
python -m playwright install chromium

# 5. Запустите GUI
python run_gui.py
```
