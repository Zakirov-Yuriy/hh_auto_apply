## 🚀 Как использовать

### Для конечного пользователя:
```bash
# 1. Запуск GUI
python run_gui.py

# 2. Или двойной клик на HH_Auto_Apply.exe (после сборки)
```

### Для разработчика:
```bash
# Установка зависимостей для разработки
pip install -r requirements-dev.txt

# Сборка exe
python build_exe.py

# Или прямая команда
pyinstaller hh_auto_apply.spec
```

## 📦 Результат

После сборки exe файл будет в папке:
```
dist/HH_Auto_Apply/HH_Auto_Apply.exe  (~150 MB)
```

Пользователи могут:
- 📥 Скачать exe файл
- ▶️ Запустить его двойным кликом
- 🔧 Настроить через GUI (не нужен Python, не нужен .env)
- 🚀 Запустить и отслеживать процесс
