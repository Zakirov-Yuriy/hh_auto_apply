## 🚀 Как Запустить

### 1. Запуск GUI (для тестирования)
```bash
python run_gui.py
```

### 2. Сборка exe (для продажи)
```bash
# Установите dev зависимости
pip install -r requirements-dev.txt

# Соберите exe
python build_exe.py
```

Результат: `dist/HH_Auto_Apply/HH_Auto_Apply.exe` (~150 MB)

### 3. Запуск exe
```bash
# Просто откройте exe файл
dist/HH_Auto_Apply/HH_Auto_Apply.exe
```
